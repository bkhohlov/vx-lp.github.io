Testing ASYNC Web Tag Hooks!

To test: click on the chat button. After approx 5 seconds, a new async welcome message will be seen in the messaging window.

      //To overwrite you can use the following code in console and then click the engagement to test this hook.
        
            lpTag.hooks.pushAsync({
              name: "BEFORE_WELCOME_MESSAGE",
              callback: function (options, successCallback, errorCallback) {
                // example asynchronous code
                setTimeout(() => {
                  options.data.content.text = "This welcome message text has been interfered with by a ASYNC hook.";
                  successCallback(options);
                }, 5000);
              }
            });
